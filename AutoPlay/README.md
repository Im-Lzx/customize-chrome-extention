## AutoPlay 插件

### 插件作用

> 优点：模拟使用者的点击事件，当一个学习视频播放完成时，自动播放学习网站的下一集视频

> 不足：对于头像采集目前无法解决，另外偶现不起作用的情况，刷新视频播放页面即可
